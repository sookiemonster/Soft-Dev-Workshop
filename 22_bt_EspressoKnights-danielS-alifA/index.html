In bootstrap, there are premade classes for rows and columns to allow easy and flexible grid-making. More specifically, .row is used to make rows in which divs with classes .col-md-n (n is any integer between 1-12). Why in between 1-12? Well, the basic grids in Bootstrap are divided into 12 equal columns, so column containers can span those preset columns.

Also, try resizing! Column elements automatically collapse into one column they can't fit.
